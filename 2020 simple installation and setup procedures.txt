1 - Install Microsoft SQL Server (make sure mixed mode to get username : sa , and password : ******)
2 - Install DotNet 4.8 (Download and install from microsoft website).
2a -Download and install DTRMSimple
3 - Start BackOffice Software and Create Database using menu (Help -> Database Administrator)
3a - Type in 
	Server IP and address
	administrator username for sql server
	administrator password for sql server
	Click connect to connect database server and list all available databases.
	Type the name for your database (default : DTRMSimple)
	Click on Create database : this will create database and required objects of the database
4 - Click on Menu (Help -> Settings and make sure you type in 
	Server IP and address
	database name you just created
	username for the database (this can be sqlserver admin username as well or create a lower level access user)
	password for the associated user.
	Ensure currency culture of the database (en-US, nl-NL, or which ever the installation language of your database server)
	Ensure currency culture of the software (which is the user interface culture exp: en-US or en-GB or nl-NL fr-FR es-ES)
	Ensure the language of the terminal (default : English)
	And..... continue with the other settings (explanation can be found for all the settings on our website)
	Click Save to close the window.
5 - Click connect on main menu to display login window and login using Admin user password : 9999
	By default system created 4 users
		Username	Password	Task
		========	========	=======
		Waiter		1		Simplest user which can use the orderpad and kitchen interfaces
		Manager		2		User which can use the orderpad and kitchen interfaces
		Admin		9999		User which can use all the interfaces and setup menu and all other objects
		Tech		2020		User has full access to system (not required normally)
	

Create Menu and Other Settings :

6 - To create menu click on Menu (Objects -> Menus -> Create Dummy [ to create dummy menu for your establisment]
	Add, delete, modify the menu accordingly
	Dummy menu creation also creates dummy Distribution [reguired to tell the system where the part of the order goes to be prepared]
	Dummy menu creation also creates dummy Categories and Category buttons, modify those based on your needs.

7 - If you want to create tables click on Menu (Objects -> Tables -> Create Dummy  [ to create dummy layout for tables])
	Add, delete, modify the table groups and their tables

8 - Complete the settings data before login to the system (Terminal based settings)
9 - Also complete the settings after login to the system (Systemwide settings)
10 - Now you can start using your system.
