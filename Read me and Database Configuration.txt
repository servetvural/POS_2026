First time installation requirements
--------------------------------------
When you installed DTRMSimple application, setup program installs all the required files and creates folders including database files and folder. However, depending on the mode this application will run on this computer; certain steps must be follewed as below.

Thin Client Mode: This computer will run as a satalite computer using other computer's DTRMSimple database.
-----------------
If this application will run as Thin Client in this computer, configuration file must be adjusted accordingly.
	- No database installation required in this computer.
	- Microsoft SQL Server (or Express edition) must be installed in another computer to connect to.
	- Installed Microsoft SQL Server (or Express edition) must have DTRMSimple database files setup and configured accordingly(please read Multi Client Server configuration below).
	- In the configuration file you must have choosen the option of ThinClient mode as the Application_Type and supply the database connection parameters(Server IP, Database name [DTRMSimple is default], username and password).

Stand Alone Mode: One computer on one site ONLY. No other computers can share the database on this computer.
-----------------
If this application will run as Stand Alone in this computer, configuration file must be adjusted accordingly.
	- LocalDB installation required in this computer(The installation link is supplied on our website http://www.dronax.com).
	- In the configuration file you must have choosen the option of StandAlone mode as the Application_Type and set true the AttachLocalDatabase File option. This local database files comes with an example menu pre-installed in it.

Multi Client Server Mode: Many computers will run DTRMSimple application (including this computer) using the DTRMSimple database on this computer through the installed SQL server on this computer.
-------------------------
If this application will run as Multi Client Server in this computer, configuration file must be adjusted accordingly.
	- Microsoft SQL Server (or Express edition) must be installed on this computer to connect to.
	- Local DTRMSimple database files must be attached to this computer's SQL Server.
	- Additional database user can be created and associated with DTRMSimple database for all the terminals to use to login locally or remotely otherwise existing dbo can be used.
	- In the configuration file you must have choosen the option of MultiClientServer mode as the Application_Type and supply the database connection parameters(Server IP, Database name [DTRMSimple is default], database username and database password).
	- The most probably you must allow remote connections to local SQL Server with creating a whole in this computer's firewall(if any).
